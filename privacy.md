
# Privacy Policy

Last updated: May 23, 2026

## 1. Information We Collect
This app does not collect, store, or share any personal information from users.

## 2. TikTok API Usage
This app uses the TikTok API solely to post content on behalf of the 
authorized TikTok account. We do not access, store, or share any TikTok 
user data beyond what is necessary to post content.

## 3. Third-Party Services
This app fetches publicly available news from VNExpress RSS feeds. 
No personal data is involved in this process.

## 4. Data Security
Since we do not collect personal data, there is no personal data at risk.

## 5. Changes to This Policy
We may update this policy from time to time. Changes will be posted in 
this document with an updated date.

## 6. Contact
For questions, contact us via GitHub Issues on this repository.
